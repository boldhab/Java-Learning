# Java Learning

Small Java console application that demonstrates basic object-oriented design and arithmetic utilities.

## Run the main app

```bash
javac src/main/java/com/example/*.java
java -cp src/main/java com.example.Main
```

## Run self-checks

```bash
javac src/main/java/com/example/*.java
java -cp src/main/java com.example.AppSelfCheck
```
